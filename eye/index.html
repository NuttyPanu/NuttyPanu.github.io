ระบบติดตามมาตรวัดน้ำ V.2

รายละเอียดการปรับปรุง V.2 : ปรับปรุงล่าสุดเมื่อวันที่ 22 ต.ค. 63

- (ฟังก์ชันค้นหา) เพิ่มการค้นหามาตรฯด้วยชุดข้อมูล CUSTCODE ( รองรับไฟล์ .xlsx )

- (ฟังก์ชันสถิติ) เพิ่มเมนูหน่วยน้ำคงที่ติดต่อกันย้อนหลัง 1- 12 เดือน (ไม่รวม 0 ลบ.ม.)

- (ฟังก์ชันสถิติ) ปรับปรุงเงื่อนไขการกรองข้อมูลให้รองรับขนาดมาตร ยี่ห้อมาตร สถานะมาตร

- (ฟังก์ชันสถิติ) ปรับปรุงเงื่อนไขการใช้น้ำ 0, 1-5, 6-10 ลบ.ม. เป็น 1- 12 เดือนย้อนหลัง

- (ฟังก์ชันแผนที่) เพิ่มการแชร์ตำแหน่งและข้อมูลในแอปพลิเคชัน Line

- (ฟังก์ชันแผนที่) เพิ่มสีอ้างอิงสถานะมาตรวัดน้ำในตารางสรุปจำนวนรายสาขา

- (ฟังก์ชันแผนที่) ปรับปรุงการแสดงผลแผนที่ (WMS) ทุกระยะการซูม

- (ฟังก์ชันแผนที่) เพิ่มมาตราส่วนแผนที่สำหรับการวัดระยะ

- (ฟังก์ชันแผนที่) เพิ่มฟังก์ชันการหาค่าระดับความสูงทั้งรูปแบบจุดและแบบเส้น (อ้างอิงค่าระดับจาก Google Map)

- (ฟังก์ชันแผนที่) เพิ่มฟังก์ชันการวิเคราะห์ข้อมูลมาตรฯในขอบเขตที่กำหนด

รายละเอียดการปรับปรุง V.1 : ปรับปรุงล่าสุดเมื่อวันที่ 3 ม.ค. 61

- เพิ่มการกำหนดสิทธิ์การใช้งานตามพื้นที่รับผิดชอบ

- (ฟังก์ชันแผนที่) เพิ่มฟังก์ชันการค้นหามาตรฯ (ค้นหามาตรฯ จาก ชื่อ, ที่อยู่ ของผู้ใช้น้ำ)

- (ฟังก์ชันแผนที่) เพิ่มฟังก์ชันการนำทางไปยังตำแหน่งของมาตรฯ (รองรับอุปกรณ์บนมือถือ)

- (ฟังก์ชันแผนที่) เพิ่มตารางแสดงรายละเอียดมาตรฯ บนแผนที่

- (ฟังก์ชันแผนที่) เพิ่มฟังก์ชันการส่งออกข้อมูลและการพิมพ์แผนที่

- (ฟังก์ชันแผนที่) เพิ่มฟังก์ชัน Google Streetview

- (ฟังก์ชันแผนที่) ปรับปรุงการแสดงผลของระบบให้รองรับอุปกรณ์บนมือถือ

- (ฟังก์ชันแผนที่) ปรับปรุงการแสดงรายละเอียดข้อมูลผู้ใช้น้ำและกราฟแสดงการใช้น้ำย้อนหลัง

- (ฟังก์ชันสถิติ) เพิ่มการกรองยี่ห้อมาตรฯและการส่งออกข้อมูล มาตรฯอายุ 10 ปี ขึ้นไป

- (ฟังก์ชันสถิติ) เพิ่มปุ่มย้อนกลับสู่แผนที่ เมื่ออยู่ในหน้าสถิติ

- (ฟังก์ชันสถิติ) เพิ่มการเลือกประเภทผู้ใช้น้ำให้สามารถเลือกประเภทย่อยได้

- (ฟังก์ชันสถิติ) ปรับปรุงการแสดงผลของกราฟและเพิ่มฟังก์ชันการส่งออกข้อมูล

ข้อกำหนดการใช้งานระบบติดตามมาตรวัดน้ำ

เงื่อนไขการใช้งาน:

    ระบบติดตามมาตรวัดน้ำจัดทำขึ้นโดยมีวัตถุประสงค์เพื่อให้บริการข้อมูลที่เกี่ยวข้องกับมาตรวัดน้ำ
ผู้ใช้งานต้องใช้งานเพื่อการปฏิบัติงานตามระเบียบและข้อบังคับของกปภ. โดยไม่เปิดเผยข้อมูล ทำให้ข้อมูลรั่วไหล หรือประกาศเผยแพร่ข้อมูลของกปภ. ส่วนใดส่วนหนึ่งหรือทั้งหมด ต่อบุคคลใด ๆ

    หากผู้ใช้งานกระทำการฝ่าฝืนข้อตกลงการใช้งาน ผู้ใช้งานจะต้องรับผิดชอบตามกฎหมายอย่างเคร่งครัด
